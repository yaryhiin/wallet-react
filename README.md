# Money Tracker App

A simple and clean React app to track personal finances.  
Manage multiple accounts in different currencies, add income/expenses, transfer money between accounts, and view transaction history.

---

## Features

- Add, edit, and delete accounts (up to 4 accounts)
- Track income and expenses per account
- Transfer money between accounts (with currency conversion)
- Default and future-customizable income/expense categories
- Validates input and highlights missing/invalid fields
- Responsive design and user-friendly UI
- LocalStorage persistence (no backend required)

## Getting Started

### Prerequisites

- Node.js (v16+ recommended)
- npm or yarn

### Installation

```bash
git clone https://github.com/yaryhiin/wallet-react.git
cd wallet-react
npm install
npm start
```

Runs the app in development mode at http://localhost:3000

---

### Usage

1. Add a new account with name, currency, and starting balance
2. Add income or expense transactions
3. Transfer money between accounts
4. View, edit, or delete transactions
5. Reset all data if needed

---

### Tech Stack

- React (with hooks)
- CSS Modules
- LocalStorage for persistence
